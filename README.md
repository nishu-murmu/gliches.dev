# My Portfolio Website

Personal portfolio website built with Astro, React, and Tailwind CSS

</div>

![Portfolio homepage](./public/website-home-screenshot.jpg)

## Tech Stack

- Astro 5
- React 19
- Tailwind CSS 4

## Local Development

```sh
pnpm install
pnpm dev
```

The dev server runs at `http://localhost:4321`.
